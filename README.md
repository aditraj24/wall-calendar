# Interactive Wall Calendar

A polished, responsive wall calendar inspired by a physical hanging calendar design.

## Features

- Date range selection
- Hover preview for range
- Notes attached to selected date ranges
- localStorage persistence
- Month navigation
- Dynamic monthly themes
- Responsive design for desktop and mobile

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- date-fns
- Framer Motion
- lucide-react

## Run Locally

```bash
npm install
npm run dev
